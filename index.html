Git & Github

Aprendendo comandos no Git

      Comandos básicos:

    • git init Inicia um repositório git na pasta atual
    • git clone URL Clona um repositório remoto para sua máquina.
    • git status Mostra o status dos arquivos no repositório.

      Branches e Navegação:

    • git branch Lista todas as branches no repositório.
    • git branch nome Cria uma nova branch.
    • git checkout branch Muda para uma nova branch.
    • git checkout -b nova-branch Cria e já troca para a nova branch

      Sincronização com Repositório Remoto:

    • git pull origin
    • git clone
