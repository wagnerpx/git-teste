Git & Github

Aprendendo comandos no Git

      Comandos básicos:

    • git init Inicia um repositório git na pasta atual.
    • git clone URL Clona um repositório remoto para sua máquina.
    • git status Mostra o status dos arquivos no repositório.

      Branches e Navegação:

    • git branch Lista todas as branches no repositório.
    • git branch nome Cria uma nova branch.
    • git checkout branch Muda para uma nova branch.
    • git checkout -b "nova-branch" Cria e já troca para a nova branch.

      Sincronização com Repositório Remoto:

    • git pull origin branch Atualiza a branch local com as mudanças do repositório remoto.
    • git pull Baixa mudanças do repositório remoto e as mescla na branch atual.
    • git push Envia suas alterações para o repositório remoto.

      Commits e Históricos:

    • git commit -m Registra mudanças com uma descrição.
    • git commit abre o editor para escrever uma mensagem mais detalhada.
    • git log Exibe o histórico de commits da branch atual.
    • git Head Indica o estado atual do repositório e a posição do último commit.
